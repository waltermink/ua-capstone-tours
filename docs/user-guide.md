# University of Alabama Tours Application User Guide

1. Firstly, open the link “https://campus-tour-backend.web.app/”, and allow location tracking.
2. From here, users can scroll and zoom on the map using scrolling and pinch and pull.
3. As you approach a location, a small info card will be displayed with the name and a brief description of the location. Users can tap this info card, to be taken to a more in depth look at the location’s information.
4. Using the different buttons on the bottom of the screen, the user can access a toolbar.
5. This toolbar allows the user to access different menus
6. The explore tab is the general access mode, where the user can view the map, have their location data allow for navigation and the small excerpts as they approach locations.
7. The List tab allows users to see all locations as a list, and clicking on a location within the list will bring up the detailed look at a location and its information.
8. The “Contribute” tab allows users to suggest new locations for the University to either approve or deny.
9. Users using “Contribute” are asked to provide a name, geographical location, short excerpt, and long form paragraph or two of information concerning the location.
